# 貢獻者行為準則

[English](CODE_OF_CONDUCT.md) · 繁體中文

## 我們的承諾

我們致力於讓任何人都能在受尊重且不受騷擾的環境中參與本專案，不因年齡、體型、身心狀況、族群、性別特徵、性別認同或表達、經驗、教育、社經地位、國籍、外貌、種族、階級、膚色、宗教、性傾向或性認同而受到差別待遇。

我們會以促進開放、友善、多元、包容且健康社群的方式互動。

## 期待行為

有助於社群的行為包括：

- 對他人保持同理與善意；
- 尊重不同意見、觀點與經驗；
- 以成熟方式提出與接受建設性回饋；
- 為錯誤負責、道歉並從中學習；
- 以專案與社群的整體利益為優先；
- 保護疑難排解過程中的 credential、私人網路資訊與個人資料。

## 不可接受行為

不可接受行為包括：

- 帶有性意味的言語、影像或不受歡迎的性關注；
- 挑釁、侮辱、貶抑言論，以及人身或政治攻擊；
- 公開或私下騷擾；
- 未經明確同意公開他人的私人資料；
- 明知有風險仍公開 credential、設備識別資訊、私人 LAN 資料或 exploit 細節；
- 持續干擾討論、issue、review 或專案基礎設施；
- 其他在專業環境中可合理認定為不當的行為。

## 適用範圍

本準則適用於所有專案空間，以及個人以官方身分代表專案的公開場合。專案空間包含 repository、issue、pull request、review、discussion 與專案相關溝通。

## 執行

若遇到濫用、騷擾或其他不可接受行為，請透過 [maintainer GitHub profile](https://github.com/zx90316) 公開的聯絡方式私下回報。含私人資訊的行為準則事件禁止使用公開 issue 回報。

所有回報會以 best-effort 方式審查與調查。Maintainer 應保護回報者隱私，並可依情況移除內容、關閉或鎖定討論、拒絕貢獻、提出警告，或暫時／永久限制參與。

## 執行原則

Maintainer 可依影響、意圖、歷史與接受回饋的態度決定處置：

1. **修正**：私下說明問題，要求修正不當言語或行為。
2. **警告**：正式警告，並說明繼續參與的條件。
3. **暫時限制**：在一定期間內限制專案互動。
4. **永久限制**：對持續或嚴重違規者永久移出專案空間。
